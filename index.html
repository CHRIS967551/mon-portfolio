01 En cours

Planning automatisé — ICFA

Outil web complet développé en alternance pour automatiser la gestion des plannings de l'école ICFA Campus Saint-François. L'application importe des fichiers CSV d'emplois du temps, attribue automatiquement les salles selon les effectifs et l'accessibilité, et affiche le planning en temps réel sur un écran TV.

Fonctionnalités clés

  • Import CSV multi-formations (MCO, NDRC, SIO, SP3S...)
  • Attribution automatique des salles selon capacité & accessibilité PMR
  • Interface admin sécurisée (login + verrouillage)
  • Affichage TV plein écran avec animation anniversaires
  • Gestion multi-années scolaires
  • Déploiement sur Raspberry Pi avec Docker + Nginx

Stack technique

Python Flask Gunicorn Docker Nginx HTML/CSS JavaScript JSON/CSV Raspberry Pi
